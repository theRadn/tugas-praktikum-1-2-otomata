## Cara Penggunaan Program
### Tokenizer / Lexer
- Masukkan source code secara manual atau upload file
- Pilih bahasa pemrograman yang digunakan
- Klik tombol "Run Lexer"
  
### FSM Checker
- Masukkan input string
- Klik tombol "Check String"
- Input result dan history akan muncul
